## Application of TDD on RESTful API using Express and Mocha framework.
### Frmaework used
- `Express 4.4.15`
- `Mocha 2.4.5`
### Testing Libraries used
- `Chai 3.5.0`
- `Chai-http 2.0.1`
### DB tools used
- `Mongoose 4.4.15`
### Instruction
> To install all the dependency
-  `> npm install`
> To run the test
-  `> mocha`
